# 빅데이터 기업 검색 시스템 

## 시스템 배경 및 목적

* 대학에서 보유하고 있는 우수 기술을 확산 및 활용하기 위한 빅데이터 기반 수요 기업 검색 서비스 필요
* 키워드를 활용한 사용자 맞춤형 수요 기업 검색 서비스의 개발

## 시스템 세부 모듈 

* Google Custom Search API 기반 키워드 관련 URL 검색 모듈
* WordNet 기반 URL 확장 모듈
* 조건식 파싱 모듈
* 기업 DB 기반 기업명 추출 모듈
* 기업명 등장 패턴 기반 기업명 추출 모듈
